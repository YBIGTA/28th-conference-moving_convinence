# 이동형 마음편의점 — 데모 UI 🏪

은평구 중장년(40~64세) 대상 **이동형 심리상담 서비스**의 사용자용 프로토타입.
고립지수 분석 결과로 도출한 방문 동선·프로그램을 주민이 직접 조회·신청하는 흐름을
[Streamlit](https://streamlit.io)으로 구현한 데모 앱이다.

> 본 폴더(`UI/`)는 앱 구동 코드만 담은 독립 단위다. 분석 파이프라인(고립지수·복지시설 등)은
> 상위 repo의 다른 폴더에 있으며, 이 앱은 그 **분석 결과를 정적 데이터(`data/`)로 내장**해 동작한다.

---

## 1. 주요 기능

| 화면 | 내용 |
|---|---|
| 로그인 / 이용 등록 | 데모용 계정 인증 (세션 기반, 별도 서버 없음) |
| 🏠 홈 | 거주 구·동 선택 |
| 📅 방문 일정 | 선택한 동의 이동형 거점 방문 요일·시간·위치 |
| 📋 프로그램 | 요일별 / 카테고리별 운영 프로그램 안내 |
| 🗺️ 노선도 | Folium 지도 — 거점·정류장·노선(요일 필터) 시각화 |
| 🤖 챗봇 | 방문 일정·프로그램 안내 (Upstage `solar-pro`) |
| 📊 분석 페이지 | "왜 은평구인가 / 고립지수란" 분석 근거 설명 |

**데모 로그인 계정** — 이름 `홍길동`, 연락처 `010-0000-0000`, 비밀번호 `0000`
(또는 "이용 등록하기"로 새 계정 생성 — 세션 동안만 유지)

## 2. 폴더 구조

```
UI/
├── app.py                 # 메인 Streamlit 앱 (단일 진입점)
├── data/                  # 분석 결과를 내장한 정적 데이터 모듈
│   ├── schedule.py        #   거점·노선·방문 스케줄
│   ├── districts.py       #   구/동 목록
│   ├── analysis_data.py   #   고립지수·우선순위·차원 데이터
│   └── programs.py        #   요일·카테고리별 프로그램
├── .streamlit/
│   └── config.toml        # 테마(색상·폰트)
├── requirements.txt       # Python 의존성
├── packages.txt           # 배포 환경 apt 패키지 (한글 폰트)
└── README.md              # 이 파일
```

---

## 3. 로컬 실행

Python 3.10+ 권장 (3.13에서 동작 확인).

```bash
cd UI
pip install -r requirements.txt
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 자동 오픈.

## 4. 챗봇 API 키 설정 (선택)

챗봇 탭은 **Upstage API 키**가 있어야 동작한다. 키가 없어도 나머지 화면(일정·프로그램·노선도·분석)은
정상 동작하며, 챗봇 탭만 오류 메시지를 표시한다.

`UI/.streamlit/secrets.toml` 파일을 만들고 키를 넣는다.

```toml
UPSTAGE_API_KEY = "up_xxxxxxxxxxxxxxxx"
```

> `secrets.toml`은 `.gitignore`에 등록돼 있어 커밋되지 않는다. **키를 코드에 직접 넣지 말 것.**

---

## 5. 배포 (Streamlit Community Cloud)

- GitHub `main`에 push하면 연결된 Streamlit Cloud 앱이 자동 재배포된다.
- 배포 환경의 챗봇 키는 Streamlit Cloud 대시보드의 **Secrets**에 `UPSTAGE_API_KEY`로 등록한다.
- `packages.txt`의 `fonts-nanum`은 서버에 한글 폰트를 설치해 지도·차트의 한글 깨짐을 막는다.
- 서버 타임존은 UTC이므로 시간 표시는 코드에서 KST로 변환해 처리한다.

## 6. 참고

- 로그인·회원가입은 **데모용**이라 실제 사용자 DB가 없다 (입력값을 세션에 저장).
- `data/` 모듈은 분석 파이프라인의 산출물을 사람이 읽을 수 있는 형태로 정리한 정적 스냅샷이다.
  분석 로직을 갱신하면 이 모듈도 함께 업데이트해야 한다.
